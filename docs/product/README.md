# SiteProof Product Source Of Truth

This folder contains curated product strategy docs that should be easier to use
than the raw research reports in `docs/research/`.

Use these files when planning features, writing QA plans, designing onboarding,
or briefing a new agent on who SiteProof is for.

| Document | Use it for |
| --- | --- |
| [user-stories.md](user-stories.md) | Target users, jobs-to-be-done, pain points, product principles, and role-based user stories. |

The raw research remains in `docs/research/` for traceability. Do not delete it
when creating curated product docs.
